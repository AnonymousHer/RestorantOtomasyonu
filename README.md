# Restorant Otomasyonu

## Proje Hakkında

Restorant Otomasyonu, restoran işletmeleri için sipariş takibi, masa yönetimi ve menü düzenlemesi gibi işlevleri içeren kapsamlı bir otomasyon sistemidir. Kolay kullanımı ve özelleştirilebilir yapısı sayesinde farklı büyüklükteki restoranlarda etkin şekilde kullanılabilir.

## Kullanılan Teknolojiler ve Diller

- **PHP**: Sunucu tarafı uygulama mantığı ve veri işlemleri.
- **CSS**: Arayüz tasarımı ve stil düzenlemeleri.
- **JavaScript**: Dinamik ve etkileşimli öğelerin yönetimi.
  
## Kullanım

1. Projeyi klonlayın:
   ```bash
   git clone https://github.com/AnonymousHer/RestorantOtomasyonu.git
   ```
2. Gerekli kurulumları ve bağımlılıkları tamamlayın.
3. Sunucu üzerinde çalıştırın (örn. XAMPP, MAMP veya Laravel Homestead).
